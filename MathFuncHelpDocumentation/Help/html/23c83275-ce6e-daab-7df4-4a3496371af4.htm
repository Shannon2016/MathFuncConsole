MathObjectInput Method

To cast an input to what is should be (a FuncTResult, whose generic type argument is Double). You should use this function in your .ctor for every parameters input to enlarge flexibility. Only types that can cast to a Double or a FuncTResult that returns a Double or null are accepted.
