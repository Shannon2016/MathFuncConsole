Bond
Initial a new Bond with following parameters, could be either static numbers or reference from other variables. I use Object as input to allow you to pass in a number(Double/Int32) or a FuncTResult(whose generic type argument is Double, as you will get from any other MathObject) or null(use default value). Types other than these three will raise an ArgumentException. You can change any of these property later on as you do in any other place of C# but be sure to wrap any static number.
